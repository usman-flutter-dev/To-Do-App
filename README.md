# To-Do App (Flutter)

A simple **To-Do application built with Flutter**.  
This project demonstrates the basic structure of a Flutter mobile app and shows how users can create and manage daily tasks in a clean and simple interface.

---

## Overview

This project was created as part of my learning journey in **Flutter app development**. The goal of the app is to practice building mobile user interfaces and understanding the structure of a Flutter project.

The application allows users to manage tasks and stay organized during daily activities.

---

## Features

- Add new tasks
- Display tasks in a list
- Simple and clean user interface
- Beginner-friendly Flutter project structure
- Easy to expand and customize

---

## Project Structure

lib/
- main.dart — Entry point of the Flutter application  
- screens/ — Application screens  
- widgets/ — Reusable UI components  

---

## Getting Started

Clone the repository and run the project using Flutter.

git clone https://github.com/usman-flutter-dev/To-Do-App.git  
flutter pub get  
flutter run  

Make sure **Flutter is installed on your system** before running the project.
